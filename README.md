# 🛡️ Suspicious Activity Detector (Python)

A Python-based cybersecurity project that detects suspicious or abnormal activity by monitoring logs, requests, or network behavior in real time.

This project was built to practice Blue Team and defensive security concepts such as intrusion detection, activity monitoring, anomaly detection, and threat analysis using Python.

---

# 📌 Features

- 🔍 Detects suspicious behavior patterns
- 📡 Monitors activity in real time
- ⚠️ Generates alerts for abnormal activity
- 📄 Logs suspicious events for investigation
- 🐍 Built completely with Python
- 🛡 Beginner-friendly cybersecurity project

---

# 🧠 Project Overview

The Suspicious Activity Detector continuously analyzes activity and attempts to identify unusual behavior based on predefined rules or suspicious patterns.

The goal of this project is to simulate the behavior of a lightweight Intrusion Detection System (IDS) or security monitoring tool.

Examples of suspicious activity include:

- Multiple repeated requests
- Unusual traffic spikes
- Unknown or blacklisted IP addresses
- Repeated failed access attempts
- Abnormal user behavior
